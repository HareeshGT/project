Problem 6: Check Prime Number

➟ This program checks if a given number is prime.

int number = 17;
boolean isPrime = true;
for (int i = 2; i <= number / 2; i++) {
    if (number % i == 0) {
        isPrime = false;
        break;
    }
}
if (isPrime) {
    System.out.println("Prime");
} else {
    System.out.println("Not Prime");
}


Problem 7: Factorial of a Number

➟ This program calculates the factorial of a given number.

int number = 5;
long factorial = 1;
for (int i = 1; i <= number; i++) {
    factorial *= i;
}
System.out.println("Factorial: " + factorial);


Problem 8: Fibonacci Series

➟ This program generates the Fibonacci series up to a given number of terms.

int n = 8;
int firstTerm = 0, secondTerm = 1;
System.out.print("Fibonacci Series: ");
for (int i = 1; i <= n; i++) {
    System.out.print(firstTerm + " ");
    int nextTerm = firstTerm + secondTerm;
    firstTerm = secondTerm;
    secondTerm = nextTerm;
}


Problem 9: Find Maximum in an Array

➟ This program finds the maximum element in an array.

int[] array = {23, 45, 12, 67, 89, 54};
int max = array[0];
for (int i = 1; i < array.length; i++) {
    if (array[i] > max) {
        max = array[i];
    }
}
System.out.println("Maximum element: " + max);


Problem 10: Check Palindrome

➟ This program checks if a given string is a palindrome.

String str = "level";
boolean isPalindrome = str.equals(new StringBuilder(str).reverse().toString());
if (isPalindrome) {
    System.out.println("Palindrome");
} else {
    System.out.println("Not a Palindrome");
}
