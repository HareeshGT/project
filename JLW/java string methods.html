String Methods in Java

1. length()

Description: Returns the length of the string.

                
        String str = "Hello, World!";
        int length = str.length(); // Returns 13

Explanation: The `length()` method returns the number of characters in the string, including spaces and punctuation.

2. charAt(index)

Description: Returns the character at the specified index.

                
        String str = "Hello, World!";
        char firstChar = str.charAt(0); // Returns 'H'

Explanation: The `charAt(index)` method allows you to retrieve the character at the specified index. In Java, indices start from 0.

3. substring(beginIndex)

Description: Returns a substring from the specified index to the end.

                
        String str = "Hello, World!";
        String sub1 = str.substring(7); // Returns "World!"

Explanation: The `substring(beginIndex)` method extracts a substring starting from the specified index until the end of the string.

4. substring(beginIndex, endIndex)

Description: Returns a substring from the specified begin index to the specified end index.

                
        String str = "Hello, World!";
        String sub2 = str.substring(0, 5); // Returns "Hello"

Explanation: The `substring(beginIndex, endIndex)` method extracts a substring from the specified begin index (inclusive) to the specified end index (exclusive).

5. concat(str)

Description: Concatenates the specified string to the end of the invoking string.

                
        String firstName = "John";
        String lastName = "Doe";
        String fullName = firstName.concat(" ").concat(lastName); // Returns "John Doe"

Explanation: The `concat(str)` method concatenates the specified string to the end of the current string.

6. indexOf(str)

Description: Returns the index of the first occurrence of the specified substring.

                
        String str = "Hello, World!";
        int index = str.indexOf("World"); // Returns 7

Explanation: The `indexOf(str)` method returns the index of the first occurrence of the specified substring. If not found, it returns -1.

7. lastIndexOf(str)

Description: Returns the index of the last occurrence of the specified substring.

                
        String str = "Hello, World!";
        int lastIndex = str.lastIndexOf("l"); // Returns 10

Explanation: The `lastIndexOf(str)` method returns the index of the last occurrence of the specified substring. If not found, it returns -1.

8. equals(str)

Description: Compares the content of two strings for equality.

                
        String str = "Hello, World!";
        boolean isEqual = str.equals("hello, world!"); // Returns false

Explanation: The `equals(str)` method compares the content of two strings character by character. It is case-sensitive.

9. equalsIgnoreCase(str)

Description: Compares the content of two strings for equality, ignoring case.

                
        String str = "Hello, World!";
        boolean isEqualIgnoreCase = str.equalsIgnoreCase("hello, world!"); // Returns true

Explanation: The `equalsIgnoreCase(str)` method compares the content of two strings for equality without considering the case.

10. toUpperCase()

Description: Converts all characters in the string to uppercase.

                
        String str = "Hello, World!";
        String upperCaseStr = str.toUpperCase(); // Returns "HELLO, WORLD!"

Explanation: The `toUpperCase()` method transforms all characters in the string to uppercase.

11. toLowerCase()

Description: Converts all characters in the string to lowercase.

                
        String str = "Hello, World!";
        String lowerCaseStr = str.toLowerCase(); // Returns "hello, world!"

Explanation: The `toLowerCase()` method transforms all characters in the string to lowercase.

12. trim()

Description: Removes leading and trailing whitespaces from the string.

                
        String spacedStr = "   Hello, World!   ";
        String trimmedStr = spacedStr.trim(); // Returns "Hello, World!"

Explanation: The `trim()` method eliminates leading and trailing whitespaces from the string.

13. replace(oldStr, newStr)

Description: Replaces all occurrences of a specified substring.

                
        String str = "Hello, World!";
        String replacedStr = str.replace("World", "Java"); // Returns "Hello, Java!"

Explanation: The `replace(oldStr, newStr)` method replaces all occurrences of the specified substring with the new substring.

14. replace(oldStr, newStr)

Description: Replaces all occurrences of a specified substring.

                
        String str = "Hello, World!";
        String replacedStr = str.replace("World", "Java"); // Returns "Hello, Java!"

Explanation: The `replace(oldStr, newStr)` method replaces all occurrences of the specified substring with the new substring.

15. split(regex)

Description: Splits the string into an array of substrings based on the specified regular expression.

                
        String sentence = "Java is a powerful programming language";
        String[] words = sentence.split(" "); // Returns ["Java", "is", "a", "powerful", "programming", "language"]

Explanation: The `split(regex)` method breaks the string into an array of substrings using the specified regular expression as the delimiter.

16. format(format, args)

Description: Returns a formatted string using the specified format string and arguments.

                
        String formattedStr = String.format("Name: %s, Age: %d", "John", 25); // Returns "Name: John, Age: 25"

Explanation: The `format(format, args)` method creates a formatted string using the specified format string and arguments.


Java String Problems

Problem 1: Palindrome Checker

                
        import java.util.Scanner;
        
        public class PalindromeChecker {
            public static void main(String[] args) {
                Scanner scanner = new Scanner(System.in);
        
                System.out.print("Enter a word or phrase: ");
                String input = scanner.nextLine();
        
                boolean isPalindrome = checkPalindrome(input);
        
                if (isPalindrome) {
                    System.out.println("The input is a palindrome.");
                } else {
                    System.out.println("The input is not a palindrome.");
                }
        
                scanner.close();
            }
        
            private static boolean checkPalindrome(String input) {
                String cleanedInput = input.replaceAll("\\s", "").toLowerCase();
                String reversedInput = new StringBuilder(cleanedInput).reverse().toString();
                return cleanedInput.equals(reversedInput);
            }
        }

Explanation: The `PalindromeChecker` class takes a word or phrase as input and checks whether it is a palindrome. - The `checkPalindrome` method removes spaces and converts the input to lowercase for a case-insensitive check. - It then uses a `StringBuilder` to efficiently reverse the string. - Finally, it compares the cleaned input with its reversed version to determine if it's a palindrome.

Problem 2: Word Frequency Counter

                
        import java.util.HashMap;
        import java.util.Map;
        import java.util.Scanner;
        
        public class WordFrequencyCounter {
            public static void main(String[] args) {
                Scanner scanner = new Scanner(System.in);
        
                System.out.print("Enter a sentence: ");
                String inputSentence = scanner.nextLine();
        
                Map wordFrequencyMap = countWordFrequency(inputSentence);
        
                System.out.println("Word Frequency:");
                for (Map.Entry entry : wordFrequencyMap.entrySet()) {
                    System.out.println(entry.getKey() + ": " + entry.getValue());
                }
        
                scanner.close();
            }
        
            private static Map countWordFrequency(String sentence) {
                Map wordFrequencyMap = new HashMap<>();
                String[] words = sentence.split("\\s+");
        
                for (String word : words) {
                    word = word.replaceAll("[^a-zA-Z]", "").toLowerCase();
                    wordFrequencyMap.put(word, wordFrequencyMap.getOrDefault(word, 0) + 1);
                }
        
                return wordFrequencyMap;
            }
        }

Explanation: The `WordFrequencyCounter` class takes a sentence as input and counts the frequency of each word in the sentence. - The `countWordFrequency` method splits the input sentence into words and removes punctuation. - It uses a `HashMap` to store the frequency of each word. - The method updates the word frequency in the map, considering case-insensitivity.

Problem 3: Email Validator

                
        import java.util.Scanner;
        
        public class EmailValidator {
            public static void main(String[] args) {
                Scanner scanner = new Scanner(System.in);
        
                System.out.print("Enter an email address: ");
                String email = scanner.nextLine();
        
                boolean isValid = validateEmail(email);
        
                if (isValid) {
                    System.out.println("The email address is valid.");
                } else {
                    System.out.println("The email address is not valid.");
                }
        
                scanner.close();
            }
        
            private static boolean validateEmail(String email) {
                String emailRegex = "^[a-zA-Z0-9_+&*-]+(?:\\.[a-zA-Z0-9_+&*-]+)*@(?:[a-zA-Z0-9-]+\\.)+[a-zA-Z]{2,7}$";
                return email.matches(emailRegex);
            }
        }

Explanation: The `EmailValidator` class validates whether a given string is a valid email address based on common email format rules. - The `validateEmail` method uses a regular expression to check if the input string matches a simple email format.

Problem 4: Password Strength Checker

                
        import java.util.Scanner;
        
        public class PasswordStrengthChecker {
            public static void main(String[] args) {
                Scanner scanner = new Scanner(System.in);
        
                System.out.print("Enter a password: ");
                String password = scanner.nextLine();
        
                int strengthLevel = checkPasswordStrength(password);
        
                System.out.println("Password Strength: " + getStrengthDescription(strengthLevel));
        
                scanner.close();
            }
        
            private static int checkPasswordStrength(String password) {
                boolean hasMinimumLength = password.length() >= 8;
                boolean hasUppercase = !password.equals(password.toLowerCase());
                boolean hasLowercase = !password.equals(password.toUpperCase());
                boolean hasDigit = password.matches(".*\\d.*");
                boolean hasSpecialChar = !password.matches("[a-zA-Z0-9]*");
        
                if (hasMinimumLength && hasUppercase && hasLowercase && hasDigit && hasSpecialChar) {
                    return 4; // Very Strong
                } else if (hasMinimumLength && hasUppercase && hasLowercase && hasDigit) {
                    return 3; // Strong
                } else if (hasMinimumLength && (hasUppercase || hasLowercase) && hasDigit) {
                    return 2; // Moderate
                } else if (hasMinimumLength) {
                    return 1; // Weak
                } else {
                    return 0; // Very Weak
                }
            }
        
            private static String getStrengthDescription(int strengthLevel) {
                switch (strengthLevel) {
                    case 4:
                        return "Very Strong";
                    case 3:
                        return "Strong";
                    case 2:
                        return "Moderate";
                    case 1:
                        return "Weak";
                    default:
                        return "Very Weak";
                }
            }
        }

Explanation: The `PasswordStrengthChecker` class checks the strength of a password based on various criteria such as length, presence of uppercase, lowercase, digit, and special character. - The `checkPasswordStrength` method evaluates the password based on these criteria and returns a strength level. - The `getStrengthDescription` method returns a descriptive label for the password strength level.

Problem 5: String Compression

                
        import java.util.Scanner;
        
        public class StringCompression {
            public static void main(String[] args) {
                Scanner scanner = new Scanner(System.in);
        
                System.out.print("Enter a string: ");
                String inputString = scanner.nextLine();
        
                String compressedString = compressString(inputString);
        
                System.out.println("Compressed String: " + compressedString);
        
                scanner.close();
            }
        
            private static String compressString(String input) {
                StringBuilder compressed = new StringBuilder();
                int count = 1;
        
                for (int i = 0; i < input.length(); i++) {
                    if (i + 1 < input.length() && input.charAt(i) == input.charAt(i + 1)) {
                        count++;
                    } else {
                        compressed.append(input.charAt(i)).append(count);
                        count = 1;
                    }
                }
        
                return compressed.length() < input.length() ? compressed.toString() : input;
            }
        }

Explanation: The `StringCompression` class compresses a given string by replacing repeated characters with the character followed by the count of repetitions. - The `compressString` method iterates through the input string, counting consecutive occurrences of the same character. - It appends the character and its count to a `StringBuilder`. - If the compressed string is shorter than the original string, it returns the compressed string; otherwise, it returns the original string.
